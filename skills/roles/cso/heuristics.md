# CSO — heuristics

Fast-lookup for strategy / GTM / competitive / partnership / big-bet decisions.

---

## The 30-second triage

1. **What kind?** Where-to-play / how-to-win / competitive / partnership / M&A / big-bet / market-entry / market-exit.
2. **What's the time horizon?** If <1 quarter, it's tactical; not really CSO. If 2-5 years, that's CSO.
3. **What's given up?** Strategy is the art of saying no. If the answer doesn't name what's given up, it's not a strategic answer.

## Where-to-play heuristics

| Question | Heuristic |
|---|---|
| Should we enter market X? | If you can't name 5-10 specific first customers, no. |
| Should we expand to a new vertical? | Only after you've fully won at least one vertical. Premature expansion dilutes everything. |
| Should we expand geographically? | Same product to a new geography is usually easier than new product to same geography. |
| Should we add a new persona / role? | Easier than new vertical. Adjacent personas often share workflows. |
| Should we move upmarket / downmarket? | Pricing wedge is risky. Move only if the unit economics make sense AND you have a clear wedge for the new tier. |
| Should we abandon a market? | If <kill-threshold metrics 9 months in, yes. Don't drag it out. |

## How-to-win heuristics

| Question | Heuristic |
|---|---|
| What's our wedge? | Specific + true + defensible for 2-3 years. If you can't fill in all three, you don't have a wedge yet. |
| Are we differentiated? | Differentiation ≠ wedge. Differentiation says "we're different"; wedge says "we're the only one who does X for Y." |
| Should we add this feature competitors have? | Only if customers are asking for it AND it doesn't compromise your wedge. |
| Can we win on price? | Almost never sustainably. Price wedges erode. Move up the stack instead. |
| Can we win on customer success? | Yes, if the segment values it AND you can deliver consistently. Most operators undervalue this. |
| What's our unfair advantage? | If you can't name a real one, you're playing a fair-fight game. That's harder than it sounds. |

## Competitive heuristics

| Trigger | Heuristic |
|---|---|
| Competitor ships X | Ignore most of the time. React only if customers materially demand it OR it changes your wedge. |
| Competitor raises Series C | Track it, but don't change strategy. Capital ≠ winning. |
| Competitor announces partnership | Investigate the implications, but don't reflexively match. |
| Competitor cuts prices 30% | Don't follow. Differentiate up. Price wars don't have winners. |
| Competitor hires someone you know | Track. Doesn't change strategy unless they're hiring an unusual specialist. |
| Three competitors are doing the same thing | Now it's a signal. Investigate what's changed in the market. |
| Customer says they're considering a competitor | Ask why. Often reveals a wedge gap or onboarding gap. |

## Partnership heuristics

| Question | Heuristic |
|---|---|
| Should we do this partnership? | Run the 4-question filter (strategic fit / distribution value / effort vs reward / reversibility). Default = no. |
| How much engineering time should we invest? | Match to expected upside. If they won't bring 100+ qualified prospects, <2 weeks of effort. |
| Should we be exclusive? | Almost never. Exclusivity is a one-way door. |
| Should we co-market? | Cheap (compared to engineering integration). Easier to say yes. |
| Should we sign a strategic alliance? | These rarely deliver. Define a specific value exchange OR don't sign. |
| Partner is moving slowly | Either the partnership isn't real or you don't have champion buy-in. Surface, decide. |

## M&A heuristics

| Question | Heuristic |
|---|---|
| Should we acquire this company? | At operator-stage, usually no. Build first. |
| Should we entertain an acquisition offer? | Hear it out — knowing your acquisition value is useful. Don't accept without M&A advisor. |
| Acqui-hire? | Maybe, if the team fills a critical gap + cultural fit + price is right (~$1-2M / engineer typical). |
| When is the right time to sell? | When you're in a position of strength (growing, profitable or near-profitable, not desperate). |
| Should we pivot vs. sell? | If you've lost the conviction in the original thesis but the team is great, pivot. If the team is also done, sell. |
| Founder-CEO is being pushed to sell by board / investors | Read the investor rights agreement; talk to other founders; don't act under pressure. |

## Big-bet heuristics

| Question | Heuristic |
|---|---|
| Is this bet worth making? | Compute expected value (probability × upside) − (1-probability × downside). If positive, consider; if asymmetric upside, consider strongly. |
| How long should we commit? | 12-24 months is the operator-stage default. Longer commitments without check-ins are gambling. |
| What's our check-in cadence? | Quarterly minimum; monthly during early execution. |
| When do we walk? | Define the kill threshold UPFRONT. After the fact, it's harder to walk away. |
| Are we being too cautious? | Maybe. The cost of NO is also real. Inaction has its own price. |
| Are we being too aggressive? | Pre-mortem the bet (Klein 2007). What would have to be true for this to fail? Surface 3-5 failure modes. |

## Pivot heuristics

| Trigger | Move |
|---|---|
| Customer feedback consistently says "wrong problem" | Investigate seriously. Maybe pivot. |
| Sales cycles getting longer in your core segment | Investigate market shift; consider pivot. |
| Founder no longer believes in the thesis | This is decision-forcing. Either re-find conviction or pivot. |
| Multiple competitors gaining ground simultaneously | Maybe the market has shifted; investigate. |
| Investor expressing concern | Listen. They see patterns. Doesn't mean they're right but doesn't mean they're wrong. |
| Strong evidence the market is bigger / different than thought | Lean in, not pivot. |

## Cognitive bias quick-catches

| Founder is saying… | Suspect this bias |
|---|---|
| "We need to do everything to win" | Strategy is saying no. Pick. |
| "Our competitors are doing X, we should too" | Sheep bias. Often wrong. |
| "This partnership will be transformative" | Optimism bias. Run the framework. Most partnerships don't deliver. |
| "We just need to execute better" | True sometimes; sometimes the strategy is wrong, not the execution. |
| "If we just had more funding" | Capital rarely solves strategic problems. Usually masks them. |
| "Our customers love us" | Maybe. Are they retaining? Expanding? Referring? That's love. Survey scores aren't. |
| "The market is huge" | TAM isn't strategy. SAM and SOM are. |
| "Let's be patient with this initiative" | Often hiding from a kill decision. Define the kill threshold. |

## When to escalate to engage mode

- Annual strategy review (full where-to-play / how-to-win doc)
- Market entry decision (detailed plan + cost ceiling + kill threshold)
- Major partnership evaluation
- Acquisition consideration
- Big-bet commitment doc
- Pivot decision

These are engage-mode artifacts because the decisions are large enough to be revisited.

---

## 80/20 of strategy cadence

```
WEEKLY (founder/CEO):
- One strategic question: "What's the one decision that matters most this week?"

MONTHLY:
- Track big-bet leading indicators
- Adjust tactically within the strategy

QUARTERLY:
- Strategy check-in (are we executing? Has the market shifted?)
- OKR / objective recalibration

ANNUALLY:
- Full strategy review
- Update where-to-play / how-to-win
- Define next year's big bets
```
