# Offer letter checklist — template

For every offer, the checklist below ensures legal cleanliness, expectation-setting, and IP protection. Operator-grade for US W-2 hires; flag jurisdictional differences for non-US.

**Risk tier:** YELLOW for routine offers; RED if the offer involves anything non-standard (custom equity terms, executive-tier severance, international employment, regulated industry).

---

## Required elements

```
□ Legal entity name (your company's exact legal name, not DBA)
□ Recipient's full legal name
□ Position title
□ Reporting line (manager name + title)
□ Start date
□ Work location (office address, remote, or hybrid policy)
□ Employment classification (full-time, part-time, exempt / non-exempt)
□ Compensation:
    □ Base salary (annual; specify if exempt/non-exempt for overtime)
    □ Variable compensation (if applicable — bonus structure, commission plan)
    □ Equity grant (number of options/shares, vesting schedule, strike price will be set by 409A)
    □ Benefits eligibility (health, dental, vision, 401(k), PTO, etc.)
□ At-will statement (US, where applicable):
    "Employment is at-will. Either party may terminate the employment 
    relationship at any time, with or without cause or notice."
□ Confidentiality + IP assignment language (referenced from separate agreement)
□ Conditions of employment (background check, reference check, eligibility to work, NDA signature)
□ Acceptance deadline
□ Signature lines (employee + company representative)
```

---

## US-specific additional elements

```
□ I-9 verification reminder (must complete within 3 days of start)
□ E-Verify notice (if applicable per state)
□ At-will statement (state-specific language for any state with non-standard requirements)
□ State-specific notices:
    □ California: notice of California Code Section 2870 (employee inventions developed on own time)
    □ Other states: similar carve-outs may apply
□ Tax form W-4 to be completed on or before start date
□ ERISA disclosures (group benefits)
□ Direct deposit form
```

---

## Equity-specific section

```
□ Equity type: ISO / NSO / RSU
□ Number of shares (e.g., 50,000)
□ Vesting schedule: typically 4 years, 1-year cliff, monthly thereafter
□ Acceleration triggers (single or double on change of control, if any)
□ Exercise window post-termination (default 90 days; some companies extend to 7-10 years for vested ISOs)
□ Reference to the company's equity plan documents
□ Note: actual strike price determined by current 409A valuation
□ Note: stock plan agreement must be executed separately
```

---

## Confidentiality + IP assignment

This is critical. The offer letter references but should NOT be the sole document for IP assignment. Use a separate Confidentiality and Invention Assignment Agreement.

```
□ Separate Confidentiality & Invention Assignment Agreement (CIIA) prepared
□ CIIA signed as a condition of employment
□ Schedule A on CIIA listing employee's pre-existing inventions
□ State-specific carve-outs included (CA, IL, KS, MN, WA all have variants)
```

Without CIIA: in most US states, the default IP-ownership rule means the employee owns the IP they create. This is the most expensive HR mistake — costs the founder cap table on departure.

---

## Benefits language

Keep this paragraph brief in the offer letter; refer to the benefits summary document for details.

```
You will be eligible for [Company]'s benefits programs, including health 
insurance, dental, vision, 401(k), and paid time off, in accordance with 
the Company's policies and the terms of the applicable benefit plans. 
Details of these benefits are described in the Employee Handbook and 
Benefits Summary that will be provided upon employment.
```

---

## Decline path

Include guidance on what happens if they don't accept:

```
Acceptance deadline: [date — typically 7-14 days from offer]

To accept: sign and return this offer letter, the Confidentiality & 
Invention Assignment Agreement, and complete the following on or before 
your start date:
- I-9 Employment Eligibility Verification
- W-4 Tax Withholding
- State tax withholding form (if applicable)
- Direct deposit authorization

If this offer doesn't meet your needs, please let us know so we can 
discuss. If we don't hear from you by [date], we'll assume you've 
declined.
```

## Non-routine elements (RED tier — get CLC review)

Some offers include non-standard provisions. These need CLC review before sending:

```
□ Signing bonus + clawback (must be clearly drafted; clawback enforceability varies)
□ Severance promise upfront (most offers DON'T promise severance; if you do, draft carefully)
□ Non-compete (largely unenforceable in CA; varies elsewhere)
□ Non-solicit (more enforceable but jurisdictional)
□ Accelerated vesting upfront (less common; usually negotiated as event-driven)
□ Specific notice period (instead of at-will, if local law requires)
□ Multi-jurisdiction provisions (employee in different state/country than entity)
```

---

## Pre-send review

```
□ Compensation numbers verified against 409A (for strike price)
□ Manager approved the offer terms
□ Recruiting / HR signed off on consistency with band
□ Legal name + entity correct
□ Start date confirmed (not a holiday, not in vacation period)
□ Acceptance deadline reasonable (7-14 days typical)
□ All referenced documents (CIIA, equity agreements, benefits summary) ready to send
□ Offer letter template last reviewed by CLC within the past 12 months
```

---

## Closing the candidate after sending

```
□ Verbal offer extended FIRST (call before sending the written letter)
□ Written letter follows within 24 hours
□ Candidate has manager / closer point of contact for questions
□ Don't pressure deadline if candidate needs to think (but do reach out at half-deadline)
□ Negotiation is expected — have your walk-away clear before sending
□ Reference checks completed before extending (or at minimum before they sign)
```

---

## After acceptance

```
□ Counter-signed offer returned to candidate
□ Onboarding process triggered (IT, payroll, equipment, accounts)
□ First-day schedule sent
□ Pre-start communications (welcome email, what to expect)
□ Calendar reminder set for 30/60/90-day check-ins
□ Equity grant approved by board (within 30 days of start typically)
□ 83(b) election filed if applicable (within 30 days of grant date — RED tier if missed)
```

---

## Templating discipline

Have ONE master offer letter template, reviewed by CLC, used for all routine hires. Variants:

```
Template A: Standard US W-2 employee (default)
Template B: Senior / Executive (with negotiated terms placeholder)
Template C: International (per country, EOR-managed)
Template D: Contractor (different document; not actually an "offer letter")
```

When you have to deviate from the template, CLC reviews the deviation. Otherwise, the deviation becomes a precedent that creates inconsistency.

---

## When to escalate to CLC

| Situation | Action |
|---|---|
| Non-standard equity terms | CLC review before sending |
| Severance promise upfront | CLC review (most operators don't promise this; if you do, draft carefully) |
| Multi-jurisdiction (employee in different state/country than entity) | CLC review for local-law compliance |
| Signing bonus with clawback | CLC review for clawback enforceability |
| Custom non-compete or non-solicit | CLC review (often weakly enforceable; rewrite or omit) |
| First international hire | CLC + international employment specialist |
| First executive (VP+) hire | CLC + compensation specialist |
