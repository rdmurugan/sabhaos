# CHRO — heuristics

Fast-lookup for hiring / classification / comp / performance / org decisions.

---

## The 30-second triage

1. **What kind?** Hire / classify / comp / performance / process / policy / employment-legal.
2. **What jurisdiction?** US (which state) / EU / UK / India / other. Different defaults each.
3. **Is this a legal-exposure question?** If yes, route to CLC; CHRO frames the people decision, CLC frames the legal exposure.

---

## Hire-vs-defer heuristics

| Question | Heuristic |
|---|---|
| Should we hire FTE? | Cost of delay > 1.5× loaded cost AND runway > 9 months AND role defined for 12+ months. |
| Should we hire contractor? | Cost of delay > loaded cost AND (runway < 9 months OR role <90% defined). |
| Should we defer? | Cost of delay < loaded cost OR runway < 6 months. |
| Is this role ready to hire? | Can you write the first-90-day deliverables in 1 hour? If no, define more. |
| First or second senior person? | First: hire for missing skill. Second: hire for missing capacity. |

## Classification heuristics

| Worker situation | Action |
|---|---|
| Uses your equipment, attends standups, full-time hours, >3 months | Employee. Restructure or convert. |
| Project-scoped, own equipment, has other clients, <30 hrs/week | Contractor. Probably OK. |
| Started as contractor, now FT for 6+ months | Reclassify (or restructure). Risk grows daily. |
| California, NY, or EU | Strictest classification rules. Default to employee. |
| Multiple workers in similar gray-area setup | Class-action exposure. CLC review now. |
| DOL / state DOL / IRS inquiry received | RED tier. Stop. Call employment attorney. |

## Comp heuristics

| Question | Heuristic |
|---|---|
| What should we pay? | 50th-75th percentile of market via Levels.fyi / Pave. Adjust by stage discount. |
| Should we match a counter-offer? | If they're a key contributor AND you can afford it AND they're not chronic-renegotiators — yes, match. Tie partial match to a future milestone. |
| Should we offer equity refresh? | After 4-year vest, yes for retained talent. Don't promise upfront; reward continued tenure. |
| Variable comp / bonus? | Sales: yes (commission). Other roles: skip until $50M+ revenue. |
| Should we adjust comp upward across the board? | Annual review process. Don't do off-cycle except for retention / promotion. |

## Performance management heuristics

| Situation | Move |
|---|---|
| One person consistently below bar | Manager 1:1 first. PIP only if fixable. Severance + exit if fit problem. |
| Manager wants to PIP — is it legit? | PIP only if (a) the issue is fixable and (b) the manager will genuinely coach. If neither, it's a pretense — don't. |
| End of PIP, employee didn't improve | CLC review severance + release. Termination meeting with HR + manager + witness. |
| Two-star manager wants to fire star performer | Pause. Often the manager is the problem. Investigate before firing. |
| Multiple complaints about one manager | Investigate. Document. May need to move the manager out of the role. |
| Layoff coming | WARN Act check (US). CLC review for discrimination analysis. Outplacement consideration. |

## Interview / hiring heuristics

| Question | Heuristic |
|---|---|
| How many interview stages? | 4-6 for routine, 5-7 for senior, 7+ for executive. Don't over-engineer for early-career roles. |
| How long should onsite be? | 3-4 hours of interviews + breaks. Beyond that, fatigue degrades signal. |
| Reference checks — how many? | 2-3 minimum for senior; 1-2 for mid; sometimes skip for entry-level. |
| Should we negotiate? | Yes. Operators who don't negotiate signal weakness. Have a clear walk-away. |
| Candidate has competing offer — what do we do? | Don't compete on price alone. Compete on opportunity, team, growth, equity upside, autonomy. |
| 90-day check — they're not working out | Don't drag it out. Have the conversation. Most "not working" doesn't get better. |

## Org design heuristics

| Question | Heuristic |
|---|---|
| Should we hire a VP for X? | Only if you have 5+ ICs in that function AND the function is the bottleneck. Below that, hire a strong senior IC, not a VP. |
| Span of control too high (>10 directs) | Add a layer. Promote a strong senior IC. |
| Too many layers (>4 for <100 people) | Flatten. Layers add coordination tax. |
| Functional or matrix? | Functional until 50+ employees. Matrix is for coordination across 2+ axes, which most operator-stage doesn't need. |
| Remote or in-office? | Pick deliberately. No right answer. Match policy to type of work + talent pool. |
| Time zone spread getting hard | Cap at 3 time zones for synchronous-heavy roles; up to 5 for async-first roles. |

## Onboarding / offboarding heuristics

| Question | Heuristic |
|---|---|
| What's good onboarding? | Day 1 ready (equipment, accounts), Week 1 ship something small, Month 1 ship something meaningful, 90-day fit decision explicit. |
| Should the founder onboard new hires? | First 5-10 hires: yes, founder owns. Beyond that: founder onboards leaders only. |
| Severance norm? | US: 2-4 weeks <1 yr, 4-12 weeks 1-3 yrs, 12-16 weeks 3-5 yrs, 16+ for executives. Always in exchange for signed release. |
| Should we communicate the firing to the team? | Yes, same day. Brief, factual, no shaming. CLC-approved messaging. |

## Cognitive bias quick-catches

| Founder is saying… | Suspect this bias |
|---|---|
| "We need to hire 5 senior engineers this quarter" | Activity bias. What's the actual work? Hire to definable scope. |
| "She's amazing but she has these issues" | Halo / horns. Separate the rating from the affection. |
| "He just needs more time / another chance" | Sunk cost. After 90-day check, decisions matter more than hopes. |
| "We should be remote because everyone else is" | Bandwagon. Match policy to work type, not industry trend. |
| "I don't want to PIP — too formal" | Avoidance. Either the issue needs PIP, or it needs termination. Don't drift. |
| "Let's pay above market to attract talent" | Often the wrong move. Market rate + great equity + clear growth path beats top-of-market salary with no upside. |
| "We're too small for HR" | True until ~10 employees. Above that, having no HR function creates exposure. |

## When to escalate to engage mode

- First hire of a critical function (eng leader, sales leader, etc.) — strategy + role definition memo
- Compensation strategy / band overhaul
- Performance Improvement Plan (formal document)
- Severance package preparation
- Layoff plan
- Hiring rubric / interview process design
- Remote / hybrid policy decision
