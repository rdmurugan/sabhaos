# Playbook — trademark registration (US-focused)

For when you want to register a trademark for a product, brand, or service. Operator-grade walkthrough — when to do it yourself, when to use an attorney, what to expect at each stage.

**Risk tier:** YELLOW for routine registration; RED if a dispute or opposition is in progress.

**Jurisdictional scope:** US (USPTO) focus. EU (EUIPO), UK (IPO), and other jurisdictions have analogous but different processes. For multi-country protection, consider the Madrid Protocol with help of an IP attorney.

---

## Step 0 — Decide whether to register

Trademark rights in the US exist from first commercial use (common-law rights). Registration adds:

- Nationwide protection (vs. common-law's geographic limits)
- Presumption of validity
- Right to use ® (vs. ™ which any common-law mark can use)
- Enforcement leverage (treble damages, attorneys' fees in some cases)
- Customs enforcement against counterfeit imports
- Constructive notice (others can't claim "I didn't know")

**Register when:**
- You're committed to the brand for 2+ years
- The brand has commercial significance (more than a placeholder)
- You're planning expansion that requires nationwide protection
- You want to license or sell the brand
- You're in a category where infringement is likely

**Don't register when:**
- The product / brand is still in heavy iteration (you might rename)
- Sub-product naming where the parent brand is what matters
- Internal codenames not used externally

## Step 1 — Trademark search (operator-handleable, ~1 hour)

Before filing, search for conflicts. Skipping this step is the most common operator mistake — you can pay the filing fee, wait 6-12 months, and then have your application refused.

### Free searches:

```
☐ USPTO TESS (https://tmsearch.uspto.gov)
   - Search for exact mark + close variants
   - Filter by your International Class (see Step 2)
   - Check both LIVE and DEAD marks

☐ Google search for the mark + your industry
   - Common-law marks not in USPTO can still block you

☐ Domain availability
   - .com is primary; secondary TLDs are also signal

☐ App Store / Google Play search for the mark in your category

☐ Social media handle availability (Twitter/X, Instagram, LinkedIn, TikTok)

☐ Web Archive (archive.org) for prior use of the mark you might be missing
```

### Red-flag patterns in your search:

| Pattern | Severity | Operator move |
|---|---|---|
| Identical live registration in your class | RED | Don't file. Different brand needed. |
| Identical live registration in a different but related class | YELLOW | Consult IP attorney before filing |
| Similar (sound-alike, look-alike) live registration in your class | YELLOW | Consult IP attorney |
| Common-law mark with verifiable first use in commerce predating yours | RED | Don't file. They have priority. |
| Dead registration (cancelled or expired) — same mark, same class | GREEN, usually — but check if abandonment was actual |
| Generic / descriptive term you can't really own | YELLOW | Consider stylized mark or rebranding |

### When to upgrade to a paid / attorney search:

- High-stakes brand (your company name vs. product feature name)
- You found 3+ borderline matches
- You're considering an unusual mark (foreign language, made-up word, etc.)
- You're entering a regulated industry where naming conflicts are common (food, drug, financial)

Cost: $300-$1500 for a comprehensive professional search; $2-5K with attorney opinion. Worth it for high-value brands.

## Step 2 — Identify your International Class

Trademarks register by class. The Nice Classification has 45 classes (34 goods, 11 services).

**Common classes for tech startups:**

| Class | What |
|---|---|
| 9 | Downloadable software, mobile apps, electronic media |
| 35 | Business management, advertising, online retail |
| 36 | Financial services, fintech, insurance |
| 38 | Telecommunications, messaging, streaming |
| 41 | Education, training, entertainment (incl. online courses) |
| 42 | SaaS, cloud computing, software development, AI services |
| 44 | Medical, healthcare |
| 45 | Legal services, online dating, security |

For a SaaS company: typically Class 42 (the SaaS itself) + Class 9 (if you have a downloadable app) + Class 35 (for any marketing / commerce features).

**Each class is a separate filing fee.** Choose classes based on actual use; don't over-file.

## Step 3 — Choose mark type

| Type | Description | When to use |
|---|---|---|
| **Standard character (word) mark** | Mark in plain text, no style/color | Strongest protection — covers any styling |
| **Stylized / design mark** | Specific logo, font, color | Protects the specific design; weaker word protection |
| **Combined** | Word + design together | Standard for full brand |

**Operator recommendation:** file the standard character mark first (it's the most protective). If you have budget, add the design mark separately for the logo.

---

## Step 4 — Choose filing basis

Two paths:

### 1A — Use in commerce (already using the mark)

You've already used the mark commercially. Faster registration. Requires:
- Date of first use anywhere
- Date of first use in interstate commerce
- Specimens (proof of use): screenshots of website / product / packaging showing the mark in use

### 1B — Intent to use

You plan to use the mark commercially but haven't yet. Reserves the mark while you launch. Requires:
- Statement of bona fide intent to use
- Later: filing a Statement of Use with proof, before registration completes
- Additional fees for the SOU + possible extensions

**Operator pick:** 1A if you're already in market; 1B if you're 6+ months from launch and want to reserve the name.

## Step 5 — File with USPTO

Two main filing options:

| Option | Fee per class | Notes |
|---|---|---|
| **TEAS Standard** | $350 | More flexible identification of goods/services |
| **TEAS Plus** | $250 | Must use predefined identifications from USPTO Trademark Manual |

For most startups, TEAS Plus is fine and saves $100/class.

**Filing requires:**
- Mark (word or image upload)
- Owner information (typically the legal entity, not the founder personally)
- Goods/services description
- Filing basis (1A or 1B)
- Specimens (1A only)
- Attorney correspondence info (US-domiciled applicants can self-file; foreign-domiciled MUST have a US attorney)

**Time to filing:** 30-60 minutes if you've already done the research.

## Step 6 — The USPTO process (6-18 months)

What happens after you file:

| Phase | Time | What happens |
|---|---|---|
| **Filing receipt** | Same day | Confirmation of submission |
| **Examination assignment** | 3-6 months | USPTO examiner reviews application |
| **Office action (if any)** | 6-9 months | Examiner raises issues — you respond within 6 months |
| **Publication for opposition** | After examiner approves | Mark published in Trademark Official Gazette for 30 days; anyone can oppose |
| **Registration (if no opposition)** | ~9-12 months after publication | For 1A filers, registration certificate issues |
| **Statement of Use (1B only)** | After publication clearance | File proof of actual use; then registration issues |

**Most likely complications:**

- **Office action requiring response** — common for first-time filers. Operator can sometimes respond to simple issues (clarifying goods description); complex issues need attorney response.
- **Refusal based on likelihood of confusion** — examiner found a conflicting mark. Counsel-driven response.
- **Refusal based on descriptiveness / genericness** — your mark is too generic to be a trademark. Counsel-driven response (sometimes successfully argued, sometimes needs rebranding).
- **Opposition during publication window** — someone files a notice of opposition. RED tier — call IP counsel immediately.

## Step 7 — When to involve an attorney

| Trigger | When |
|---|---|
| Filing strategy (multiple marks, multiple classes, complex priority) | Before filing |
| Foreign filing strategy (Madrid Protocol vs. national filings) | Before filing |
| Refusal based on likelihood of confusion | When office action issues |
| Refusal based on descriptiveness | When office action issues |
| Opposition filed during publication | Immediately |
| Opposition you might file against someone else | When the conflicting mark publishes |
| Trademark infringement letter (incoming or outgoing) | RED tier always |
| International filings | For each foreign jurisdiction |
| Enforcement / litigation | RED tier always |

**Cost expectations (US, 2026 rates, rough):**
- Self-filing simple registration: $250-350 per class (USPTO fee only)
- Attorney-filed simple registration: $1,500-3,500 per class (fees + attorney time)
- Office action response: $500-2,500 depending on complexity
- Opposition or cancellation proceeding: $10,000-50,000+
- Infringement litigation: $100,000+ easily

## Step 8 — Post-registration maintenance

Once registered, you must:

```
☐ Use the mark continuously in commerce (non-use for 3+ years presumed abandonment)
☐ File Section 8 Declaration of Continued Use — between 5th and 6th anniversary
☐ File Section 9 Renewal — every 10 years
☐ File Section 15 Declaration of Incontestability (optional but valuable) — 
   between 5th and 6th anniversary
☐ Monitor for infringement (search alerts, customs registration if relevant)
☐ Enforce when you find infringement (silence is bad for the mark)
☐ Update USPTO records on any ownership change (acquisition, restructuring)
```

**Set calendar reminders 6 months before each deadline.** Missing Section 8 = mark cancelled. Missing Section 9 = mark expires.

## Quick decision tree

```
Brand worth protecting AND no obvious conflicts AND budget for $250-500 in fees?
├── YES → Self-file via TEAS Plus (operator-grade)
├── Some uncertainty → Pay for a paid search before self-filing
├── Found borderline conflicts → Consult IP attorney before filing
└── Brand is high-stakes (company name, multi-product, international) → 
    IP attorney throughout
```

## Special situations

### Stylized vs. word mark — file which first?

Word mark first. It protects the brand regardless of how you visually style it. Add the stylized mark later if budget allows.

### Domain name vs. trademark — same thing?

No. Owning the domain doesn't give you trademark rights. Owning the trademark doesn't give you the domain.

For a full brand strategy:
1. Trademark search first
2. Secure the trademark (or at least confirm clear path)
3. Buy the domain (.com primary, secondary TLDs depending on budget)
4. Reserve social handles

### What if you want to use a competitor's brand keyword in ads?

Generally allowed under trademark law (competitive keyword advertising). But:
- Don't use their mark in your ad text or landing page in confusing ways
- Don't use their mark in your domain
- Don't claim affiliation with them

Counsel review for any aggressive campaign.

### Geographic / descriptive marks — the hardest category

Marks like "California Bakery" or "Quick Coffee" or "Best Software" are descriptive and often unregistrable on the Principal Register. You can sometimes register on the Supplemental Register (less protection) and convert to Principal after 5 years of use.

**Operator move:** avoid descriptive marks unless you have a strong reason. Made-up words ("Spotify," "Xerox," "Kodak") or arbitrary words ("Apple" for computers) are the strongest mark types.

### Stylized marks for descriptive or weak word marks

If your brand name itself is too descriptive to register as a word mark, you can sometimes register a stylized version (the specific logo). Protection is narrower (covers the specific visual style) but it's something.
